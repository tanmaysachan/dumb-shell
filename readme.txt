Basic shell
does basic shell stuff

1) ls, cd, echo, pwd, history, pinfo, exit and execvp handled in shell_functions.h

2) Error handling done through error_handler.h

3) function lookups go in function_lookup.h

4) utils.c contains utility functions used by the shell.

5) shell.c contains code to initialise and run the shell.

6) run "make" to compile the shell.

7) execute the shell by running "./a.out".
